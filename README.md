wap
===

web app. go fast.

* orchid - docker orchestration tooling and config
* rocker - rails 4 api, devise, no sprockets
* frozen - angular, browserify, gulp
* nginx - nixxer
* puma - fastcat
* postgresql
* redis
* elasticsearch
* sidekiq - worker_bee


```sh
git clone https://github.com/lazyproof/wap
orchid dev
vagrant up
```
